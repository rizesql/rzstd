\begin{frame}{Literals Section}
  \begin{itemize}
    \item \textit{Literalii} reprezintă simboluri emise explicit, neacoperite de un \textit{match}
      prin mecanismul de look-back.

    \item Sunt codificați prin \textbf{Huffman coding}.

    \item Pentru eficiență, secțiunea poate fi imparțită în până la \textbf{4 fluxuri
      independente}, decodificate separat, dar utilizând același tabel de ponderi.

    \item Tabelul de ponderi poate fi, la rândul său, codificat prin FSE.
  \end{itemize}
